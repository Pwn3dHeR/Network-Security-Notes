## QEMU软件包：

获取文件信息：

```
.\qemu-img.exe info '源文件'
```

常用参数： -p 显示进度  -f 转换 -c 压缩

Vmdk —> Qcow2

转换命令：`qemu-img convert -f vmdk -O qcow2 source-name.vmdk target-name.qcow2  `

转换&压缩命令：`qemu-img convert -c -f vmdk -O qcow2 source-name.vmdk target-name.qcow2 `

转换&压缩&进度 命令：`qemu-img convert -p -c -f vmdk -O qcow2 source-name.vmdk target-name.qcow2 `

Qcow2 —> Vmdk

```
.\qemu-img.exe convert -f qcow2 -O vmdk '源文件' '目标文件.vmdk’
```
