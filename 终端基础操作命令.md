## Powershell

------

### **1. 文件与目录操作（PowerShell 特有方式）**

- `Get-ChildItem`（`ls` / `dir` 的增强版）  

```powershell
Get-ChildItem -Recurse       # 递归列出所有子目录内容
Get-ChildItem -Filter "*.txt"  # 按条件筛选文件
```

- `New-Item`（创建文件或目录）  

```powershell
New-Item -Path "C:\log.txt" -ItemType File     # 创建文件
New-Item -Path "C:\NewFolder" -ItemType Directory  # 创建目录
```

- `Remove-Item` **的增强功能**  

```powershell
Remove-Item -Recurse -Force C:\OldFolder  # 强制删除目录（包括只读文件）
```

------

### **2. 系统信息与管理（PowerShell 特有）**

- `Get-Process`（`ps`，比 `tasklist` 更强大）  

```powershell
Get-Process | Where-Object { $_.CPU -gt 10 }  # 筛选 CPU 占用 >10% 的进程
```

- `Stop-Process`（精准终止进程）  

```powershell
Stop-Process -Name "notepad" -Force  # 强制终止记事本
Stop-Process -ID 1234               # 根据 PID 终止
```

- `Get-Service`（服务管理）  

```powershell
Get-Service | Where-Object { $_.Status -eq "Running" }  # 查看运行中的服务
Start-Service -Name "WinRM"         # 启动服务
Stop-Service -Name "WinRM" -Force   # 强制停止服务
```

------

### **3. 网络相关（PowerShell 特有）**

- `Test-NetConnection`（替代 `ping`，更详细）  

```powershell
Test-NetConnection google.com -Port 80  # 测试端口连通性
```

- `Invoke-WebRequest`（`curl` / `wget` 的替代）  

```powershell
Invoke-WebRequest -Uri "https://example.com" -OutFile "page.html"  # 下载网页
```

- `Get-NetIPConfiguration`（比 `ipconfig` 更结构化）  

```powershell
Get-NetIPConfiguration | Select-Object InterfaceAlias, IPv4Address  # 只看网卡和 IP
```

------

### **4. 数据处理与管道（PowerShell 核心特性）**

- `Where-Object`（`?`，条件过滤）  

```powershell
Get-Process | Where-Object { $_.WorkingSet -gt 100MB }  # 过滤内存占用 >100MB 的进程
```

- `Select-Object`（`select`，选择特定属性）  

```powershell
Get-Service | Select-Object Name, Status  # 只显示服务名和状态
```

- `Export-Csv` **/** `Import-Csv`（数据导出导入）  

```powershell
Get-Process | Export-Csv -Path "processes.csv"  # 导出为 CSV
$data = Import-Csv -Path "processes.csv"       # 导入 CSV
```

------

### **5. 其他实用命令**

- `Get-Command`（查找命令）  

```powershell
Get-Command *service*  # 查找包含 "service" 的命令
```

- `Get-Help`（详细帮助）  

```powershell
Get-Help Get-Process -Examples  # 查看命令使用示例
```

- `Get-Member`（`gm`，查看对象属性）  

```powershell
Get-Process | Get-Member  # 查看进程对象的所有属性和方法
```

------

### **PowerShell 独有特性**

1. **面向对象**：所有命令返回结构化对象（非纯文本）。  

```powershell
(Get-Process notepad).Kill()  # 直接调用方法终止进程
```

1. **管道强化**：支持传递对象而非文本。  

```powershell
Get-Service | Where-Object { $_.Status -eq "Stopped" } | Start-Service
```

1. **别名系统**：兼容 Unix/Linux 习惯（如 `ls`、`curl`）。  

```powershell
ls | ? { $_.Length -gt 1000 }  # 等效于 Get-ChildItem | Where-Object
```

------

### **快速区分 PowerShell vs CMD**

| 场景     | CMD 命令     | PowerShell 命令                 |
| -------- | ------------ | ------------------------------- |
| 终止进程 | `taskkill`   | `Stop-Process`                  |
| 服务管理 | `sc`         | `Get-Service` / `Start-Service` |
| 网络测试 | `ping`       | `Test-NetConnection`            |
| 下载文件 | 需第三方工具 | `Invoke-WebRequest` (内置)      |

PowerShell 的优势在于 **对象化处理** 和 **跨平台兼容性**（支持 Windows/Linux/macOS）。
